# Multi-System Character Manager
『Trump TRPG Systems』というTRPGシステムのキャラシート管理用のツールとして作っていたものを汎用システムに作り替えました。

# 『Trump TRPG Systems』とは
『Trump TRPG Systems』とはトランプまたは1d14と1d4の2つのダイスを用いる私が考案したTRPGシステムです。
以下のような特徴があります。
・基礎ステータス4つに割り振れる数値は合計30で1つのステータスには最大13まで割り振り可能
・経験値50ごとに装備や技能のスロットを1個解放する
・トランプまたは1d14と1d4で判定を行う
・(トランプの数字または1d14)<=ステータスで成功判定を行う、14またはジョーカーはファンブルとする
・判定成功時にステータスに割り振られている数字またはスートと1d4の数字またはトランプのスートが一致したらクリティカルとする
・4つの基礎ステータスは共通キャラデータとして他の『Trump TRPG Systems』採用システムに相互にコンバート可能とする
(概念的には手塚治虫などのスターシステムみたいなもので世界観でキャラの立ち位置とかは変わりはする)
・システムの拡張としてポーカーやブラックジャックによる特殊な判定も実装可能とする
